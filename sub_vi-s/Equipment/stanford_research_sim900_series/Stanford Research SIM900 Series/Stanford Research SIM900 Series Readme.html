3. Known Issues

To report issues or provide feedback about this instrument driver, please contact technical support.

Extraneous Errors Returned Form Instrument

Note firmware tested during development SIM900: ver3.6; SIM928: ver2.2; SIM960: ver2.17; SIM965: ver4.18

Some Commands for the SIM928 and SIM965 produce execution errors even when they execute properly. These error have been masked using the Mask Error private VI. This should be fixed in a new firmware version.

In the Configure PID control VI for the SIM960, a no change execution error is produced if there is no change to which elements are enabled in the PID control path. This error has been masked using the mask error private VI.
