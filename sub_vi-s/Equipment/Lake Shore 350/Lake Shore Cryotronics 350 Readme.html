Lake Shore Cryotronics 350 Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Lake Shore Cryotronics
Supported Language(s): LabVIEW
Supported Model(s): 350
Model(s) Tested: 350
Interface(s): GPIB, Serial, Ethernet
Firmware Revision(s): Main Version 1.5, Input Version 1.3

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 1.0.2
Original Release Date: 07/2013
Current Revision Date: 07/2016

2. Required Software

Some software components need to be installed before using this instrument driver. Minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 5.0 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please contact technical support. Each VI in this driver includes time delays as required by the instrument for stable communication.
Please refer to the Message Flow Control section in Chapter 6 of the Model 350 User’s Manual for detailed information
